# AI Effectiveness
AI can dramatically accelerate development but still requires human validation and oftentimes does not catch it's own errors. I had to manually screen my code for duplicate classes and syntax errors. AI helped a great deal in this project by:
1. Generating repetitive boilerplate code quickly
2. Producing consistent class structures
3. Assisting with JUnit test creation
4. Accelerating documentation generation

## Code Quality Compariason
Although I used to AI applications in tandem with each other ChatGPT is the better AI for:
1. Minor formatting improvements
2. Additional validation logic
3. More descriptive JavaDoc comments

## Learning Experience What I Learned
Some of my most important takeaways from this assignment were:
1. Inheritance allows multiple classes to share common behavior.
2. Polymorphism allows objects to be treated through base-class references.
3. Prestaging boilerplate code on a gaming mouse will save minutes if not hours of development time.
4. When running test be wary of duplicate classes which will cause compilation errors.
